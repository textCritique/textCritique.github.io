Sierpinski Gasket

The Sierpinski gasket is a recursively defined region. To draw it, we need to know the coordinates of the initial triangle and the order of the gasket.

Midpoint Calculation

The only prerequisite is to know that the coordinate of the midpoint of a line segment (in our case, the sides of the triangle) is the average of the coordinates of the endpoints defining the line segment.

For example, if AB is a line segment and M is its midpoint, where A ≡ (x1, y1) and B ≡ (x2, y2), then:

M ≡ ( (x1 + x2) / 2, (y1 + y2) / 2 )

Algorithm

Sierpinski-gasket (x1, y1, x2, y2, x3, y3, n)

  1. If n = 0, then:
    • Draw a filled triangle defined by points A, B, and C.
  2. If n > 0, then:
    • Draw Sierpinski gasket defined by A, D, and F, reducing order by 1.
    • Draw Sierpinski gasket defined by D, B, and E, reducing order by 1.
    • Draw Sierpinski gasket defined by F, E, and C, reducing order by 1.
